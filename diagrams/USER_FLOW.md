## Map Encoding

**Color = Coverage Level:**
- рҹҹҘ Red (0-25%) = Critical gap
- рҹҹ§ Orange (25-50%) = High risk  
- рҹҹЁ Yellow (50-75%) = Moderate
- рҹҹ© Green (75-100%) = Adequate

**Pattern = Data Quality:**
- в–Ҳв–Ҳв–Ҳв–Ҳ Solid = High confidence (trust вүҘ70)
- в–“в–“в–“в–“ Stripe = Low confidence (trust <70)

**Examples:**
- рҹҹҘ Solid = Confirmed gap
- рҹҹҘ Stripe = Possible gap, data too sparse
